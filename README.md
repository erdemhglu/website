# erdemhglu/website

Source of [erdemhacisalihoglu.com](https://erdemhacisalihoglu.com) — my personal site, portfolio, and blog.

## Stack

- [Next.js](https://nextjs.org/) (App Router, Turbopack) + React
- TypeScript
- Tailwind CSS
- [next-themes](https://github.com/pacocoursey/next-themes) for light/dark mode
- Markdown blog posts (`gray-matter`, `remark`, `rehype`)

## Features

- Bilingual content (German / English), with the default picked from the visitor's browser locale
- Light/dark theme, following the OS preference by default
- Markdown-based blog with syntax highlighting and an RSS feed
- Project portfolio with a photo gallery/lightbox

## Development

```bash
npm install
npm run dev
```

```bash
npm run build
npm start
```
